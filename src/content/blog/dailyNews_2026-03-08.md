### [MySQL 9.6重构外键约束机制：解决CDC数据一致性问题](https://www.infoq.cn/article/sRyWIACcuxPivTo8DwdA)

来源：InfoQ 推荐

发布时间：2026-03-08 10:00:00

背景：MySQL 长期存在的架构缺陷导致外键级联操作在存储引擎层执行，未被二进制日志记录，引发 CDC 数据一致性问题。方案：MySQL 9.6 将外键约束与级联操作的管理权从 InnoDB 移至 SQL 层。这一改动使得级联变更被完整记录，确保了复制精度和数据一致性。测试表明，该架构调整未对性能造成退化。结论：此次更新修复了困扰 MySQL 许久的 CDC 管道数据盲点，显著提升了数据库在混合负载环境下的可靠性，是 CDC 用户的必选升级。

### [GCDWebServer: iOS/macOS 嵌入式 HTTP 服务器](https://github.com/swisspol/GCDWebServer)

来源：iOS Development News - Telegram Channel

发布时间：2026-03-08 10:07:43

GCDWebServer 是一款专为 iOS、macOS 和 tvOS 应用设计的现代轻量级嵌入式 HTTP 1.1 服务器。它基于 Grand Central Dispatch (GCD) 构建，具有无第三方依赖、架构优雅（仅含 4 个核心类）和高并发性能的特点。该库提供了丰富的内置功能，包括异步请求处理、大文件流式传输、Web 表单及 JSON 解析、HTTP 压缩以及身份验证。此外，它还包含 GCDWebUploader 和 GCDWebDAVServer 等扩展，方便实现基于浏览器的文件传输和 WebDAV 服务，并自动处理 iOS 应用的前后台模式切换。GCDWebServer 为移动开发者提供了一个强大且易用的解决方案，用于在应用内部署高效的本地 Web 服务。
