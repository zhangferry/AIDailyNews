### [Swift Lambda Runtime V2 发布：全新特性与迁移指南](https://swifttoolkit.dev/posts/lambda-v2)

来源：SwiftToolkit.dev

发布时间：2025-08-07 08:00:00

Swift Server Work Group 上周宣布了 AWS Lambda Runtime 的 V2 测试版发布。此版本在内部结构上进行了重写，主要特点包括背景执行、流式响应和服务生命周期支持。开发者现在可以使用 @main 入口点和块式 API 进行灵活开发，背景执行允许在响应后继续处理任务，流式响应可分块发送数据以提升用户体验。此外，服务生命周期支持使得服务管理更为有效。总的来说，此版本为开发者带来了更高的灵活性和功能。
